# Used Car Price Prediction

An end-to-end machine learning project for predicting used car resale prices using regression models, hyperparameter tuning, model evaluation, and Streamlit deployment.

## Project Overview

This project was developed to analyze the factors that influence used car resale prices and build a machine learning model capable of predicting estimated vehicle prices.

The project uses a dataset containing 10,000 used car records and 12 variables, including vehicle age, mileage, engine capacity, fuel type, ownership history, transmission, service history, accident records, and insurance status. :contentReference[oaicite:0]{index=0}

## Objectives

- Predict used car resale prices
- Identify the most influential factors affecting resale value
- Compare multiple regression models
- Optimize model performance through hyperparameter tuning
- Develop an accessible prediction application using Streamlit
